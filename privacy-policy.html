개인정보처리방침 (Privacy Policy)

최종 수정일: 2026-04-14

1. 수집하는 정보

ObsiCapture는 다음 정보를 사용자의 기기(chrome.storage.sync)에만 저장합니다.

  • Claude API 키
  • Make Webhook URL (선택)
  • Google Drive 액세스 토큰 (선택)
  • Obsidian vault 폴더명 (선택)

위 정보는 Anthropic 또는 Google 서버에 직접 전송되며, 개발자 서버로 수집되지 않습니다.

2. 정보 사용 목적

  • Claude Haiku API 호출 (대화 요약 생성)
  • Google Drive API 호출 (노트 파일 저장 및 파일 목록 조회)
  • Make Webhook 전송 (선택한 저장 방식에 따라)

3. 제3자 공유

ObsiCapture는 사용자 데이터를 개발자 또는 제3자에게 전송하거나 판매하지 않습니다. AI 대화 내용은 요약을 위해 Anthropic API로 전송됩니다.

4. 데이터 보관

모든 설정 정보는 사용자의 Chrome 프로필에 저장되며, 확장프로그램 삭제 시 함께 삭제됩니다.

5. 문의

문의사항이 있으시면 Chrome Web Store 지원 이메일로 연락해주세요.
