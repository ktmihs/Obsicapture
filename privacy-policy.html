개인정보처리방침 (Privacy Policy)

최종 수정일: 2026-04-17

ObsiCapture("확장프로그램")는 사용자의 개인정보를 중요하게 생각합니다. 본 방침은 확장프로그램이 수집, 사용, 저장, 공유하는 데이터에 대해 설명합니다.

1. 수집하는 데이터

데이터 유형 항목 수집 목적 저장 위치
인증 정보 Claude API 키 AI 요약 생성 호출 사용자 기기 (chrome.storage.sync)
웹사이트 콘텐츠 Claude.ai / ChatGPT 대화 내용 노트 요약 생성 사용자 기기 (로컬 vault에만 저장)
사용자 설정 Vault 폴더 경로, 저장 설정 노트 저장 위치 지정 사용자 기기 (IndexedDB, chrome.storage)

2. 데이터 사용 방법

  • Claude API 키: Anthropic API 호출 시 인증에만 사용됩니다.
  • 대화 내용: 요약 생성을 위해 Anthropic API로 전송되며, 이후 사용자의 로컬 Obsidian vault에 저장됩니다.
  • 설정 데이터: 확장프로그램 기능 제공에만 사용되며 외부로 전송되지 않습니다.

3. 데이터 공유

ObsiCapture는 수집한 사용자 데이터를 제3자에게 판매하거나 공유하지 않습니다. 단, 아래의 경우 데이터가 외부 서비스로 전송됩니다:

  • Anthropic API (api.anthropic.com): 대화 요약 생성을 위해 대화 내용이 전송됩니다. Anthropic의 개인정보처리방침이 적용됩니다.

4. 데이터 저장 및 보관

  • 모든 설정 데이터는 사용자의 Chrome 프로필에 로컬로 저장됩니다.
  • 대화 내용은 요약 생성 후 사용자의 로컬 Obsidian vault에 저장됩니다.
  • 확장프로그램 서버에는 어떠한 사용자 데이터도 저장되지 않습니다.
  • 확장프로그램 삭제 시 모든 로컬 데이터가 함께 삭제됩니다.

5. 데이터 보안

  • API 키는 Chrome의 암호화된 storage.sync에 저장됩니다.
  • 모든 API 통신은 HTTPS를 통해 암호화됩니다.
  • 대화 내용은 요약 생성 목적으로만 사용되며 서버에 보관되지 않습니다.

6. 사용자 권리

  • 확장프로그램 설정 페이지에서 언제든지 API 키 및 설정을 삭제할 수 있습니다.
  • 확장프로그램을 삭제하면 모든 저장된 데이터가 제거됩니다.

7. 아동 개인정보 보호

본 확장프로그램은 13세 미만 아동을 대상으로 하지 않으며, 아동의 개인정보를 의도적으로 수집하지 않습니다.

8. 방침 변경

본 개인정보처리방침이 변경될 경우, GitHub 레포지토리를 통해 공지됩니다. 변경된 방침은 게시 즉시 효력이 발생합니다.
